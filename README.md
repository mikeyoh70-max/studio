# Rekber Nusantara - Website Resmi

Website ini adalah platform landing page untuk jasa Rekening Bersama (Rekber) terpercaya di Indonesia.

## Fitur & Desain Saat Ini
- **Brand**: Rekber Nusantara (PT Rekber Nusantara).
- **Tema**: Mixed Elegant (Perpaduan Hero Gelap dan Konten Putih Bersih).
- **Efek**: Tombol RGB Gaming dengan teks Putih Bold + Shadow (Legendary UI Look).
- **Blog Edukasi**: 5 Artikel profesional tentang keamanan transaksi digital.
- **Mobile Friendly**: Carousel blog dengan *peek effect* (geser kartu) di HP.
- **Biaya**: 2% untuk transaksi di atas Rp 1.000.000.
- **Support**: WhatsApp Official ("hallo admin , saya mau Buat Rekber Sekarang Dong").

## Cara Update (Push ke GitHub)
Setiap perubahan yang lu buat harus di-push ke GitHub biar otomatis di-deploy:
1. `git add .`
2. `git commit -m "Pesan perubahan lu di sini"`
3. `git push origin main`

*Tips: Jika terminal minta password, gunakan **Personal Access Token (PAT)** GitHub lu.*

## Kontak & Verifikasi
Pastikan hanya menghubungi nomor resmi yang tertera di website ini untuk menghindari penipuan.
