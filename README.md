# Rekber Nusantara - Website Resmi

Website ini adalah platform landing page untuk jasa Rekening Bersama (Rekber) terpercaya di Indonesia.

## Fitur & Desain Saat Ini
- **Brand**: Rekber Nusantara (PT Rekber Nusantara).
- **Tema**: Mixed Elegant (Perpaduan Hero Gelap dan Konten Putih Bersih).
- **Efek**: Tombol RGB Gaming dengan teks Putih Bold + Shadow (Legendary UI Look).
- **Mobile Friendly**: Dioptimalkan 100% untuk akses dari smartphone (responsif).
- **Floating WhatsApp**: Tombol melayang dengan efek pulse untuk mempermudah akses chat admin.
- **Biaya**: 2% untuk transaksi di atas Rp 1.000.000.
- **Support**: Terhubung langsung ke WhatsApp Admin Resmi dan email `rekbernusantara777@gmail.com`.

## Cara Update (Push ke GitHub)
Setiap perubahan yang di-push ke branch utama di GitHub akan otomatis di-deploy melalui Firebase App Hosting.
1. `git add .`
2. `git commit -m "Pesan perubahan lu di sini"`
3. `git push origin main`
