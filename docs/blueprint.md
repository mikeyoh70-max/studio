# **App Name**: Safeguard Trades

## Core Features:

- Admin Verifier: Allow users to input a WhatsApp number and verify if it is a verified admin by checking against the Firestore 'admin_list' collection. Display a green checkmark or red cross with a verification message based on the result.
- Fee Calculator: Enable users to input a transaction amount and automatically calculate the Rekber fee, displaying it in a transparent pricing table.
- Trust Indicators: Showcase three key points: 'Secure (Identity Verification),' 'Fast (Minutes Count),' and 'Neutral (Law Student)' to build trust.

## Style Guidelines:

- Primary color: Deep blue (#2E5CB6) to convey trust and security, reminiscent of established financial institutions.
- Background color: Light blue (#D2E2F4) to create a calm and trustworthy atmosphere. Its low saturation provides a clean backdrop that helps the primary and accent colors pop.
- Accent color: Gold (#FFD700) to convey a premium feel and highlight important elements such as the CTA buttons, but will not be overused as to create imbalance.
- Body and headline font: 'Poppins' for a modern and gamer-friendly aesthetic (sans-serif).
- One-page scroll layout with a clear hierarchy to guide users through the information efficiently.
- Subtle Framer Motion animations for smooth transitions and an enhanced user experience.
